## Команда head
Команда head в Kali Linux (и Linux в целом) используется для быстрого просмотра начальной части файлов или вывода (по умолчанию — первые 10 строк).
Она входит в состав GNU Core Utilities, что делает её незаменимой для работы с логами и текстовыми данными в терминале. 

### Основные возможности и примеры использования:
Просмотр начала файла:
head file.txt — выводит первые 10 строк файла file.txt.
Указание количества строк (-n):
```                                                                                         
┌──(max㉿kali)-[~/work/DZ]
└─$ head -n 5 file11
123123
```
```                                                                                     
┌──(max㉿kali)-[~/work/DZ]
└─$ history | head  
    1  whoami
    2  uname -r
    3  pwd
    4  sudo apt update
    5  sudo apt upgrade -y
    6  uname -r
    7  whoami
    8  sudo reboot now
    9  apt-get install git
   10  sudo apt-get install git
```
```                                                                                         
┌──(max㉿kali)-[~/work/DZ]
└─$ tail file11
Указание количества строк (-n):
head -n 5 file.txt — выводит первые 5 строк файла.
Просмотр начала нескольких файлов:
head file1.txt file2.txt — выводит заголовки с именами файлов и их первые 10 строк.
Использование с конвейером (|):
history | head — показывает 10 самых старых команд в истории.
Вывод байтов (-c):
head -c 100 file.txt — выводит первые 100 байт файла.
Исключение последних строк (-n -):
head -n -20 file.txt — выводит весь файл, кроме последних 20 строк. 
```
                                                                                      
```                                                                                       
┌──(max㉿kali)-[~/work/DZ]
```
